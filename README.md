## О проекте

`fastapi-order-microservices` — это учебный pet-project, который демонстрирует построение backend-системы на микросервисной архитектуре.

В проекте планируется реализовать:
- `auth_app` — аутентификация, пользователи, роли и права доступа;
- `catalog_app` — товары и работа с остатками;
- `order_app` — создание и обработка заказов;
- взаимодействие между сервисами через RabbitMQ;
- отдельные базы данных PostgreSQL для каждого сервиса.

Основная цель проекта — показать практические навыки разработки backend-приложений на Python с использованием FastAPI, Docker, PostgreSQL, RabbitMQ и ролевой модели доступа.
